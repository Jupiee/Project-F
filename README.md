### **What is this project?**
This is planned to be a football/soccer simulation in terminal. It is currently in the developement stage and alot of things could be changed.

### **What is already done?**
Currently the project can only generate player profiles and their stats data. The data is completely randomized but could be recreated with a seed.

* [ ] Season, Leagues, Teams, Cups and Tournament system
